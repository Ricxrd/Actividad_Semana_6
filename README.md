# Sistema de Gestión de Pedidos - Repostería Dulce Sabor

Aplicación web desarrollada con Flask para la gestión de pedidos.

## Funcionalidades
- Registrar pedidos
- Listar pedidos
- Marcar como entregado
- Eliminar pedidos

## Tecnologías
- Python
- Flask
- SQLite
- HTML/CSS

## Ejecución
pip install flask
python app.py
